# Systemic Hypertension Lab — Interactive Cardiology Website

A self-contained, offline-first learning website built from the uploaded **Systemic Hypertension** chapter and organized around current guideline principles.

## Open

1. Extract the ZIP.
2. Open `index.html` in a modern browser.
3. No installation, package manager, web server, or internet connection is required.

## Included

- 21 interconnected learning modules
- Dual ESC and AHA/ACC blood-pressure classification calculator
- Measurement-error simulator and office technique checklist
- White-coat, masked, sustained, and normal-pattern explorer
- Mechanism map and secondary-cause clue matcher
- Target-organ damage and investigation selectors
- Lifestyle planner, drug-class explorer, and comorbidity selector
- Resistant-hypertension sequence and hypertensive-emergency triage lab
- Kidney, renovascular, and cardiorenal syndrome interactives
- Eight clinical reasoning cases
- 37 true flip flashcards
- 25-question scored quiz with explanations
- Search, bookmarks, visited progress, dark mode, print view, keyboard support, and per-section text-to-speech
- Original source PDF in `assets/systemic-hypertension-source.pdf`

## Text-to-speech

The site requests an English UK browser voice and prefers a UK female voice when one is installed. Available voices depend on the operating system and browser.

## Clinical scope

Educational use only. Diagnostic thresholds and treatment targets depend on the guideline framework, measurement method, comorbidity, tolerance, pregnancy status, and local clinical protocol.
