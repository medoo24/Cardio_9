# Systemic Hypertension Lab — Fixed Edition

Open `index.html` in a modern browser. The site works offline.

## Folder layout

All required files are deliberately kept together in this single folder:

- `index.html`
- `styles.css`
- `content.js`
- `app.js`
- `systemic-hypertension-source.pdf`
- `README.md`

The flashcards use accessible button controls, support mouse click and Enter/Space, and include Reveal all and Reset cards actions.
